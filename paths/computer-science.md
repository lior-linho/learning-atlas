# Computer Science Learning Path

## Goal

Build practical and theoretical computer science foundations for software development, scientific computing, data analysis, and research tools.

## Why This Matters

Computer science provides the tools to build reproducible systems, process large datasets, and turn scientific questions into computational workflows.
