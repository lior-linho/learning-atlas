# MIT OpenCourseWare

This folder tracks selected MIT OpenCourseWare courses used in my learning path system.

MIT OCW is used as a structured course layer for building foundations in mathematics, physics, computer science, and computational methods.

## Role in Learning Atlas

- `paths/` defines the long-term learning routes.
- `courses/mit-ocw/` tracks concrete MIT courses and study progress.
- `notes/`, `problem-sets/`, `readings/`, and `logs/` store the actual learning outputs.

## Main Areas

- Mathematics
- Physics
- Computer Science
- Computational Methods
