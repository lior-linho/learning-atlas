# Part A: Definition and Basic Rules

This part follows MIT OCW 18.01SC Single Variable Calculus, Unit 1: Differentiation.

## Purpose

Build the basic language and techniques of differentiation.

This part starts from the definition of the derivative and then develops the main differentiation rules used throughout single-variable calculus.

## Core Ideas

- Derivatives describe local change.
- The derivative is defined through a limit.
- Differentiation rules make derivative computation efficient.
- Limits and continuity explain when derivatives exist or fail to exist.
- Product, quotient, and chain rules are essential tools for compound functions.
