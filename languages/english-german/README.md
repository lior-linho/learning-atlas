# English–German Academic Vocabulary

This folder tracks English–German vocabulary pairs used in mathematics, physics, astrophysics, computer science, and university study.

The goal is to connect English learning resources such as MIT OpenCourseWare with German university-level terminology.

## Main Areas

- Mathematics
- Physics
- Astrophysics
- Computer Science
- Daily academic language

## Usage

Each file records vocabulary in this format:

| English | German | Notes |
|---|---|---|
| term | term | meaning, usage, or context |
